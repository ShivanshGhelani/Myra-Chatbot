This directory is for storing screenshots of the MyraChatBot application.

Suggested screenshots to include:
1. Main chat interface
2. Text summarization feature
3. Visual scene description feature
4. Mobile responsive view
5. Code highlighting example

When taking screenshots:
- Use a clean, professional environment
- Capture meaningful interactions with the AI
- Ensure the UI is clearly visible
- Consider different screen sizes for responsive design demonstration

Name your screenshots descriptively, for example:
- chat-interface.png
- summarization.png
- scene-description.png
- mobile-view.png
- code-highlight.png
